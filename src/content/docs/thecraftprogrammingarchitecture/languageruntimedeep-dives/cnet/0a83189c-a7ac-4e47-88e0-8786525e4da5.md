Led by programming expert Jeffrey Richter, a longtime consultant to the Microsoft .NET team, this resource offers an in-depth exploration of the common language runtime, C#, and .NET development. It aims to equip developers with pragmatic insights necessary for building robust, reliable, and responsive applications and components.

The content is fully updated for .NET Framework 4.5 and Visual Studio 2012, providing a thorough grounding in the .NET Framework architecture, runtime environment, asynchronous programming, and the new Windows Runtime. Developers will benefit from extensive code samples in Visual C# 2012 and authoritative guidance on challenging development concepts such as generics and threading.
